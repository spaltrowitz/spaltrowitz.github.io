# Shari Paltrowitz - Professional Portfolio Website

A modern, responsive portfolio website showcasing professional skills, projects, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean design with gradient backgrounds and smooth animations
- **Project Showcase**: Features three main projects:
  - Habit Rewards - A gamified habit tracking application
  - Slotted AI - An AI-powered scheduling application
  - Madison Gardens - A real estate management platform
- **Skills Display**: Organized technical skills in Frontend, Backend, and Tools categories
- **Professional Navigation**: Easy-to-use navigation with smooth scrolling
- **Contact Links**: Direct links to LinkedIn and GitHub profiles

## Technologies Used

- HTML5
- CSS3 (with animations and responsive design)
- Modern web standards

## Viewing the Website

Simply open `index.html` in any modern web browser, or deploy to any static hosting service.

## Local Development

To view locally, you can use a simple HTTP server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
